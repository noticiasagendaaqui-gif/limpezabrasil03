
ARQUIVOS PARA INSTALAÇÃO NO CPANEL - LIMPABRASIL
==================================================

CONTEÚDO DESTA PASTA:
- Todos os arquivos necessários para hospedar o site no cPanel
- Documentação completa de instalação (INSTALACAO_CPANEL.md)

INSTALAÇÃO RÁPIDA:
1. Faça upload de todos os arquivos desta pasta para public_html
2. Configure o banco de dados no cPanel
3. Edite config/database.php com suas credenciais
4. Importe database/schema.sql no phpMyAdmin
5. Configure config/email.php com suas informações de email

ARQUIVOS IMPORTANTES:
- INSTALACAO_CPANEL.md = Documentação completa
- config/database.php = Configuração do banco
- config/email.php = Configuração de email
- database/schema.sql = Estrutura do banco

SUPORTE:
Consulte INSTALACAO_CPANEL.md para instruções detalhadas
